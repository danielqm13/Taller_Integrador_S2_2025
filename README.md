# Taller Integrador S1 2026
Repositorio para Taller Integrador S1 2026

Proyecto: Diseño de desarrollo para plataforma de interconexion y prueba
para el microcontrolador Siwa utilizando metodologia agil SCRUM. 

## 1.Metodologia SCRUM:

### 1.1 Vision del producto:
Se desea diseñar y fabricar una PCB de pruebas para el microcontrolador SIWA,
debera ser capaz de controlar y correr distintos programas de pruebas para el
SIWA (Microcontrolador basado en RISC-V de baja potencia para aplicaciones 
medicas por el ITCR)
